# CTP 行情数据处理 — 移除调试打印

## 修改的需求

### 需求：移除调试打印代码

`OnRtnDepthMarketData()` 回调函数中不应包含任何硬编码的合约调试打印逻辑。

#### 场景：收到 ag2603 行情 tick

- **WHEN** CTP 行情回调收到 ag2603 的 tick 数据
- **THEN** 不输出任何调试信息到 stdout
- **AND** 行情数据正常转换并推送到共享内存队列

#### 场景：收到其他合约行情 tick

- **WHEN** CTP 行情回调收到任意非 ag2603 合约的 tick 数据
- **THEN** 处理行为与之前完全一致（无变化）
